Tecnologias usadas

Para este trabajo se utilizó Java para crear toda la lógica de negocio, también creamos los métodos necesarios para poder crear el archivo JSON con el que luego trabajamos con Angular en el frontend. Para la maquetación utilizamos Bootstrap, que ya nos provee algunos componentes reutilizables. Luego con Node.js, además de instalar todas las dependencias utilizadas desde el frontend , configuramos la aplicación para que podamos desplegarla en Heroku que nos provee servicio de hosting gratuito.
